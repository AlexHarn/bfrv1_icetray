.. _STTools:

STTools
=======

The STTools project provides a programming framework for developing IceCube hit
selection algorithms based on space (S) and time (T) hit information.

Using this framework specific ST algorithms has been developed and are
provided through STTools (see below).

This project is maintained by Martin Wolf <mwolf@icecube.wisc.edu>.
