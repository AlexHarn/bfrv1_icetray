ST Configuration Services
=========================

.. module:: icecube.STTools.seededRT.configuration_services

The seededRT algorithm of STTools provides a specialized STConfigurationService
class named ``I3SeededRTConfigurationService``. Based on that class a seededRT
configuration service class named
:ref:`STTools_algorithms_seededRT_I3DOMLinkSeededRTConfigurationService` is
provided.
