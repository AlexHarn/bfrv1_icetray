Portia
======

This project is maintained by Aya Ishihara <aya@hepburn.s.chiba-u.ac.jp>.

This module extract recorded photoelectron information in a simple way and store the information in PortiaEvent class.
Utility to split a PortiaEvent into several q frames is included.
