**Author**: Kevin Meagher

Overview
========

This project implements probability distribution functions (PDFs) for light
arriving at a DOM in IceCube. These are analytic approximations which are
suitable for reconstructing muons using the :ref:`Gulliver` framework.
It is used for the Pandel bases reconstructions SPE1st and MPE.
