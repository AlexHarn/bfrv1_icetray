.. _IceHive:

IceHive
=======

The IceHive project contains an advanced EventSplitter and Hit-Cleaning, which describes the IceCube detector in its hexagonal structure.

This project is maintained by Marcel Zoll <mzoll@icecube.wisc.edu>.
